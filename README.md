# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Deployment

When deploying this application to a hosting provider like Vercel, you will need to set the following environment variable for the AI features to work:

- `GEMINI_API_KEY`: Your API key for Google AI Studio.

This key is necessary for the AI-assisted activity logging to connect to the generative model.
